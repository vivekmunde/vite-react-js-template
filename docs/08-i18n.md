# Internationalization

## Setup

- Library: i18next + react-i18next
- Language detection: `i18next-browser-languagedetector`
- Backend: `i18next-http-backend` (loads from `/public/locales/`)

## Supported Languages

- `en`: English
- `nl`: Dutch

## Translation Files

Located in `public/locales/{lang}/`:

- `auth.json`: Authentication
- `common.json`: Common terms
- `delete.json`: Delete actions
- `error.json`: Error messages
- `no-result.json`: Empty states
- `organizations.json`: Organization terms
- `permission.json`: Permissions
- `revoke.json`: Revocation
- `validation.json`: Validation messages

## Usage

- `Translation` component: `<Translation value="key" />`
- `useLanguage` hook: `const [language, setLanguage] = useLanguage()`
- API URL includes language prefix: `{apiUrl}/{language}`

## Provider

`I18nProvider` initializes i18next with language detection and backend loader.
